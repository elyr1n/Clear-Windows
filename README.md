### English

**Windows Cleanup Script**

This batch script helps clean temporary files, crash dumps, recent documents, NVIDIA cache, recycle bin contents, prefetch data, and resets a specific registry key related to the Compatibility Assistant. Use with caution as it permanently deletes files.

---

### Русский

**Скрипт очистки Windows**

Этот батч-скрипт помогает удалить временные файлы, дампы сбоев, недавние документы, кеш NVIDIA, содержимое корзины, данные Prefetch и сбросить определенный ключ реестра, связанный с Compatibility Assistant. Используйте осторожно - файлы удаляются безвозвратно.
