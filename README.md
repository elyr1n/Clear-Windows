### English

**Windows Cleanup Script**

This batch script helps you to delete temporary files, crash dumps, recent documents, NVIDIA cache, Windows Defender history, Recycle Bin content, Prefetch data and reset a specific registry key related to Compatibility Assistant. Use with caution - files are permanently deleted.

---

### Русский

**Скрипт очистки Windows**

Этот батч-скрипт помогает удалить временные файлы, дампы сбоев, недавние документы, кеш NVIDIA, историю Windows Defender, содержимое корзины, данные Prefetch и сбросить определенный ключ реестра, связанный с Compatibility Assistant. Используйте осторожно - файлы удаляются безвозвратно.
